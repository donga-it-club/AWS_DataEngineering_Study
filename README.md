## 스터디 방식

- 동아대 도서관을 통한 온라인 교재를 바탕으로 공부하면서, 책 내용을 요약한 블로깅 또는 실습을 번역하여 자기 리포지토리에 올린 것을 발표한다.
- 매주 수요일 오후 7시에 줌으로 진행하고, 매주 3명씩 발표한다.
- 캠을 키고 스터디에 참여한다.
- AWS 기초 교육 공개는 교수님의 논의 후에 결정.
